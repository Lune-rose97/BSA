# Описание сущности "ClientDiscount" (Скидки клиенту)

## 1. Назначение сущности

Сущность хранит информацию о персональных скидках для клиентов. Менеджер создает записи для назначения скидок постоянным клиентам или в рамках акций. Система автоматически применяет скидки при бронировании услуг и расчете стоимости. Назначение сущности определяется требованиями из Use Cases:

- **UC03**: Забронировать слот на обслуживание - идентификация клиента для записи

## 2. Атрибуты сущности

### 2.1 Основные атрибуты

| Наименование | Ключ | Тип данных | Ограничение длины | Обязательность | Значение по умолчанию | Комментарии |
|-------------|------|------------|-------------------|----------------|---------------------|-------------|
| discount_id | PK | INT | - | Обязательно | AUTO_INCREMENT | Первичный ключ, уникальный идентификатор скидки |
| client_id | FK | INT | - | Обязательно | - | Внешний ключ, уникальный идентификатор клиента |
| discount_type  | - | VARCHAR | 25 | Обязательно | - |  Тип скидки (постоянный клиент, акционная и т.д.)  |
| discount_amount | - | DECIMAL  | - | Обязательно | - | Размер скидки: % или сумма  |
| start_date | - | DATE | - | Обязательно | - | Дата начала действия скидки |
| end_date | - | DATE | - | Необязательно | - | NULL | Дата окончания действия скидки |
| status | - | VARCHAR | 20 | Обязательно | 'Active' | Статус активности скидки |
| created_by | - | INT | - | Обязательно | - | Внешний ключ к сущности "Сотрудники" |

# Описание сущности "ClientNotification" (Извещение клиента)

## 1. Назначение сущности

Сущность хранит историю уведомлений, отправленных клиентам. Система автоматически создает записи при отправке напоминаний о записи, изменении расписания или специальных предложениях. Менеджер может просматривать историю уведомлений для контроля коммуникации. Назначение сущности определяется требованиями из Use Cases

- **UC03**: Забронировать слот на обслуживание — отправка подтверждения бронирования клиенту 

## 2. Атрибуты сущности

### 2.1 Основные атрибуты

| Наименование | Ключ | Тип данных | Ограничение длины | Обязательность | Значение по умолчанию | Комментарии |
|-------------|------|------------|-------------------|----------------|---------------------|-------------|
| notification_id | PK | INT | - | Обязательно | AUTO_INCREMENT | Первичный ключ, уникальный идентификатор |
| client_id | FK | INT | - | Обязательно | - | Внешний ключ, уникальный идентификатор клиента |
| notification_type | - | VARCHAR | 20 | Обязательно | - | reminder-напоминание, reschedule-перенос, promo-акция, system-системное  |
| contact_channel  | - | VARCHAR | 20 | Обязательно | - | Канал связи для уведомления |
| subject | - | VARCHAR | 255 | Необязательно | NULL | Тема сообщения |
| sent_time | - | DATETIME | - | Необязательно | NULL | Фактическое время отправки сообщения |
| send_status | - | VARCHAR | 20 | Обязательно | Отправляется | Статус отправки сообщения |
| booking_id | - | INT | - | Необязательно | NULL | Внешний ключ к сущности "Бронирования" |
| created_at | - | DATETIME | - | Обязательно | CURRENT_TIMESTAP | Дата и время создания записи |
