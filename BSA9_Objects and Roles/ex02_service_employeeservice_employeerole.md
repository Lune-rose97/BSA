# Описание сущности справочников «Service» (Услуги)

## 1. Назначение сущности

Сущность "Service"  является сущностью для хранения перечня услуг, предоставляемых барбершопом

## 2. Атрибуты сущности

### 2.1 Основные атрибуты

| Наименование | Ключ | Тип данных | Ограничение длины | Обязательность | Значение по умолчанию | Комментарии |
|-------------|------|------------|-------------------|----------------|---------------------|-------------|
| service_id | PK | INT | - | Обязательно | AUTO_INCREMENT | Первичный ключ, уникальный идентификатор услуги |
| name | - | VARCHAR | 25 | Обязательно | - | Название услуги |
| desription | - | VARCHAR | 250 | Обязательно | - | Описание услуги |
| duration | - | INT | - | Обязательно | - | Длительность услуги |
| price | - | DECIMAL | - | Обязательно | - | Стоимость услуги |
| is_active | - | BOOLEAN | - | Обязательно | FALSE | Активность в системе |
